Portofolio Saya

Tomat.In - Iot Pemilah Tomat Berdasarkan Tingkat Kematangan

Tomat In adalah aplikasi monitoring IoT yang dirancang untuk melakukan pemilahan tomat berdasarkan tingkat kematangannya. Aplikasi ini memungkinkan pengguna untuk memperoleh data berat tomat yang dikategorikan ke dalam tiga kelompok: matang, setengah matang, dan mentah. Dikembangkan menggunakan framework Flutter dan database MySQL, aplikasi ini terintegrasi dengan sistem IoT yang terdiri dari ESP32-CAM, Arduino, Load Cell, Servo, dan Konveyor. Sensor ESP32-CAM digunakan untuk mendeteksi kematangan tomat berdasarkan citra visual, sementara load cell mengukur berat setiap tomat sebelum diarahkan ke kategori yang sesuai menggunakan servo dan konveyor otomatis.

PasarTaniKediri - MarketHub Produk Pertanian di Kab. Kediri

PasarTaniKediri adalah sebuah website marketplace yang dirancang untuk memasarkan produk pertanian di Kabupaten Kediri. Platform ini membantu petani dan kelompok tani dalam menjual produk mereka dengan lebih luas dan mudah dijangkau oleh pembeli. Melalui website ini, penjual dan pembeli dapat terhubung secara langsung menggunakan tautan WhatsApp dan lokasi di Google Maps yang diunggah oleh pengguna. Hal ini memungkinkan transaksi yang lebih cepat dan transparan tanpa perantara. Website ini dikembangkan menggunakan framework Laravel dan didukung oleh database MySQL.

KaRe (Kartoharjo Recycle) - Mengelola TPST Kartoharjo Nganjuk

Kartoharjo Recycle (KaRe) adalah aplikasi berbasis mobile dan website yang dirancang untuk mengelola Tempat Pengolahan Sampah Terpadu (TPST) di Kartoharjo, Nganjuk. Aplikasi ini bertujuan untuk mempermudah pengelolaan tabungan pelanggan serta layanan TPST lainnya. Pada aplikasi mobile, admin TPST dapat menginput tabungan pelanggan secara langsung, sementara website digunakan untuk berbagai layanan seperti pemantauan tabungan, permintaan kunjungan, dan pengelolaan sampah. Aplikasi mobile dikembangkan menggunakan framework Flutter, sedangkan website dibangun dengan Laravel

Unjuk - Aplikasi Mobile dan Website Pendataan UMKM se-Kab. Nganjuk

Unjuk adalah aplikasi market hub yang dirancang untuk mendukung UMKM di Kabupaten Nganjuk dalam memasarkan produk mereka. Aplikasi ini tersedia dalam dua platform, yaitu mobile dan website, dengan fungsi yang saling terintegrasi. Pada aplikasi mobile, pengguna UMKM dapat mengunggah data produk mereka secara langsung, sehingga memudahkan proses pemasaran. Sementara itu, website digunakan untuk menampilkan produk yang telah diunggah serta berfungsi sebagai platform monitoring bagi admin. Aplikasi mobile dikembangkan menggunakan Android Studio, sedangkan untuk pengelolaan data, aplikasi ini menggunakan database MySQL

MbahSirr - Aplikasi Kasir dan Laporan berbasis Desktop

Mbahsirr adalah aplikasi berbasis desktop yang dirancang untuk mendukung operasional kasir sekaligus pembuatan laporan keuangan. Aplikasi ini memberikan kemudahan dalam pencatatan transaksi, manajemen penjualan, serta penyusunan laporan secara sistematis. Dikembangkan menggunakan NetBeans, Mbahsirr memanfaatkan database MySQL untuk menyimpan dan mengelola data dengan aman serta efisien. Dengan fitur yang terintegrasi, aplikasi ini membantu pengguna dalam mengelola transaksi secara lebih cepat dan akurat.
